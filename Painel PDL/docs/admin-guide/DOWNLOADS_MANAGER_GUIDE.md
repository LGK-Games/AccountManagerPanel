# Gerenciador de Downloads (Patchers e Clients)

> **Última atualização:** 21/02/2026

O Lineage 2 é pesado e frequentemente seus servidores mudarão. Um patcher (Launcher) precisa ter espelhos de download infalíveis. A Central de Downloads serve para o Administrador controlar Links Dinâmicos na página Inicial e estatísticas públicas de banda.

---

## 📥 Criando um Espelho (Mirror)

1. Vá até `/admin/downloads/manager/` na rota gerencial.
2. Adicione os **Provedores de Nuvem**. O sistema já vem iconografado para os pilares: 
   - Google Drive
   - Mediafire
   - Mega.nz
   - Yandex
3. Categorize se o link trata-se de um `Client Ouro (Completo)` pesando dezenas de GBs, ou de um modesto `System Patch` (zipzinho) para correções rápidas de Data. 

## ⚖️ Distribuição de Tráfego

Para que um botão "Baixe Já" não derrube seu servidor:
- **Priorize Links Premium**: Arraste os links Mega.nz para baixo caso sofram restrição de banda e coloque servidores FTP ou Google Drive primeiro.
- Desative links quebrados sem precisar programar uma linha do HTML ou mexer nos botões do site.
- Configure descrições que avisem a data: "Patch v1.9 - Extraia na pasta do L2". O Frontend injeta esse aviso lindo e dinâmico sobrepondo as capas no painel deslogado dos players novos.
