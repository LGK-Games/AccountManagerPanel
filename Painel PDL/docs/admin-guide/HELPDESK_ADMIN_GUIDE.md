# Gerenciamento de Chamados (Helpdesk)

> **Última atualização:** 21/02/2026

Este guia orienta Administradores (Staff) sobre como monitorar, responder e gerenciar de ponta a ponta as solicitações de suporte abertas pelos jogadores no PDL.

## 📥 Recebendo um Chamado

Quando um jogador abre um ticket, o sistema cria uma solicitação com status padrão de **"Aberto"**.
Para começar a triagem:
1. Navegue até o **Menu Administrativo** e acesse a aba **Solicitações** (ou acesse a URL `/admin/solicitation/solicitation/`).
2. Utilize os filtros laterais para procurar por categorias urgentes como `Bug` ou problemas atrelados a `Faturamento`.

## 🚦 Trocando Prioridades e Status

Você não apenas responde, mas mapeia o peso do problema. Ao clicar sobre um ID de chamado:
- **Nível de Prioridade**: Você pode escalar um chamado que entrou como "Média" para "Urgente" ou "Crítica". Mutações de prioridade disparam destaques na dashboard para os outros admins ficarem em alerta.
- **Mudança de Status (Workflow)**:
  - Mude para **"Em Andamento"** quando começar a investigar o caso.
  - Mude para **"Aguardando Usuário"** caso peça um log ou print e precise que o jogador responda.
  - Finalize com **"Resolvido"** ou **"Rejeitado"** ao término.

## 💬 Interagindo com o Jogador (Histórico)

1. Desça até a área de **Histórico de Comentários** (Interações).
2. Escreva sua resposta ou solicitação de dados, marcando o checkbox `Público` caso a resposta deva ser enxergada no painel do próprio jogador, e use `Interno` se for apenas um rascunho de aviso visível exclusivamente para você e o resto da equipe da Staff.
3. Se um usuário estiver insatisfeito, o ticket trará um badge informando a classificação (Rating) dada ao término.

## 🤖 Atendimento Prévio pela IA

Lembre-se que caso o módulo do **Assistente Virtual** esteja rodando em sua base de configurações, é provável que tickets com "Categoria Perfeita" tenham sido sugeridos diretamente pela inteligência artificial após falha em encontrar ajuda nas FAQs públicas, o que significa que o usuário já cruzou os tutoriais básicos antes de submeter o formulário.
