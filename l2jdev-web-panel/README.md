# ⚔️ L2JDev Web Panel

Modern web platform for Lineage II Interlude servers.

Includes a complete landing page + player control panel with real-time data integration.

---

## 🌐 Features

- 🎮 Landing page with server presentation
- 👤 Player account panel
- 🧝 Character selection and stats display
- 📊 Game statistics (PvP, PK, time online, etc)
- 💰 Donation system integration
- 🛒 Shop system
- ⚙️ Services (VIP, rename, etc)
- 📡 Real-time server status (Login/Game)
- 🌍 Multi-language support (PT-BR / EN)
- 🎨 Modern UI (dark + gold theme)

---

## 🧱 Technologies

- PHP (Backend)
- MySQL (Database)
- HTML5 + CSS3
- JavaScript
- Custom API integration with L2J server
